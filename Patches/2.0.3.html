Accessibility Auditor Release Notes: Version 2.0.3

A list of featured updates to the HTML_CodeSniffer "Accessibility Auditor" bookmarklet in release version 2.0.3, released on 15th December 2014. This list is focused for users of the bookmarklet, and focuses on changes to tests and the auditor interface. A full list of updates can also be found in the project's technical bookmarklet changelog.

WCAG 2.0 Standards

Items referring to Success Criteria and Guidelines should be read in conjunction with the W3C's documentation for the Web Content Accessibility Guidelines (WCAG) 2.0.

Standards updates

  • Success Criteria 1.3.1, 4.1.2: Version 2.0.2 updated handling of labels to form inputs, however some other tests were not changed to remove requirements that labels be associated via the "for" attribute. Those seeing "H44" and "H91" errors on wrapped labels for this reason should no longer see them.
